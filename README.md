# Hiring Test
This repository contains solutions for a three-part test assignment, covering REST API development,
task automation with Celery/Docker, and basic ML integration.

## Project Overview
The repository is organized into three main modules:

Task 1: To-Do List API – A FastAPI-based CRUD service with in-memory storage and unit tests.

Task 2: Automation & External API – A data processing script integrated with JSONPlaceholder, orchestrated by Celery and Redis within Docker containers.

Task 3: ML Priority Classifier – A simple text classification pipeline and an API endpoint to predict task priorities.

## Tech Stack
Python 3.12.12

FastAPI

Task Queue: Celery + Redis

Containerization: Docker & Docker Compose

ML Libraries: Scikit-learn, Pandas

Testing: Pytest
